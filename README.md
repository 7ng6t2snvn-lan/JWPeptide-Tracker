# Shot Tracker v6.1.5

Peptide injection tracking app with 121 peptide database.

## What's New in v6.1.5
- Added 24 new peptides from Project Biohacked community
- Database expanded from 97 to 121 peptides
- New GLP-1 alternatives (Cagrilinitide, Survodutide)
- Enhanced cognitive peptides (NA-Semax-Amidate, N-Acetyl Selank)
- Additional bioregulators and healing peptides

2. Your existing data (schedules, logs, inventory) will be preserved automatically
3. Clear browser cache after deploying
